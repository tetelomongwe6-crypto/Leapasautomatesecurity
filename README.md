# LEAPAS Automate Security - Professional Website

Modern, premium, high-converting website for LEAPAS Automate Security.

## Features

✅ Responsive Design (Mobile-First)
✅ Dark Blue, Red, White, Black, Metallic Silver Color Scheme
✅ Smooth Animations & Transitions
✅ SEO Optimized
✅ High-Converting CTAs
✅ Fast Loading Performance
✅ Sticky Navigation & WhatsApp Integration
✅ Service Showcase
✅ Customer Testimonials
✅ Contact Forms & Quote Requests

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion (Animations)
- React Icons
- Swiper (Testimonials Carousel)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build & Deploy

```bash
npm run build
npm start
```

## Deploy on GitHub Pages

1. Update `next.config.js` with GitHub Pages settings
2. Run `npm run build && npm run export`
3. Push to `gh-pages` branch
